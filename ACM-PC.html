澳大利亚投票 (Australian Voting)

PC/UVa 题号: 110108/10142

澳大利亚投票系统要求选民们将所有候选人按愿意选择的程度排序，一张选票就是一个排序。一开始，每张选票的首选项将被统计。若有候选人得票超过 50%，他将直接胜出；否则，所有并列得票最低的候选人出局，而那些将出局候选人排在第一位的选票将被重新统计为排名最高的未出局候选人。这一筛选过程将持续进行，直到某个候选人得到超过 50% 的选票，或所有候选人得票相同。

输入

输入第一行包含一个正整数，代表测试数据组数。然后是一个空行，接下来是数据。每两组测试数据间用空行隔开。

每组数据第一行为一个整数 n ≤ 20，代表候选人数目。接下来 n 行按顺序给出每个候选人的名字，最长 80 个字符，可以是任何可打印字符。接下来是至多 1000 行，每行描述一张选票的内容。每张选票包含 1 到 n 的某一排列。第一个为首选，第二个次选，以此类推。

输出

对于每组数据，输出数据可能是单独一行，即胜出者的名字；或者若干行，分别包含所有平局的候选人。每两组数据间用一个空行隔开。

样例输入

1 3 John Doe Jane Smith Jane Austen 1 2 3 2 1 3 2 3 1 1 2 3 3 1 2

样例输出

John Doe

#include <iostream>

using namespace std;

//represents a single ballot with ranked preference
class Ballot
{
public:
    vector<int> preferences;   //candidate indices in order of preference(1-based -> 0 -based)
    int currentRank;  //the current active preference index

    Ballot(const vector<int>& prefs) : preferences(prefs), currentRank(0) {}
};

//run one round of preference voting
//candidates list of candidate names
//ballots list of all ballots
//eliminated set of eliminated candidate indices
//return pair of (winner index, isTie)

pair<int, bool> runRound(const vector<string>& candidates, vector<Ballot>& ballots, set<int>& eliminated)
{
    int totalVotes = ballots.size();;
    map<int, int> voteCount;

    //initialize vote count for active candidate
    for (int i = 0; i < candidates.size(); i++)
    {
        if (eliminated.find(i) == eliminated.end())
        {
            voteCount[i] = 0;
        }
    }

    //distribute votes based on current active preference
    for (auto& ballot : ballots)
    {
        // Find the first non-eliminated candidate in the ballot
        while (ballot.currentRank < ballot.preferences.size())
        {
            int candidate = ballot.preferences[ballot.currentRank] - 1; // Convert to 0-based
            if (eliminated.find(candidate) == eliminated.end())
            {
                voteCount[candidate]++;
                break;
            }
            ballot.currentRank++;
        }
    }

    // Check for majority winner
    for (auto& [candidate, count] : voteCount)
    {
        if (count > totalVotes / 2)
        {
            return { candidate, false };
        }
    }

    // Find candidates with the minimum votes
    int minVotes = totalVotes;
    for (auto& [candidate, count] : voteCount)
    {
        if (count < minVotes)
        {
            minVotes = count;
        }
    }

    // Eliminate all candidates with minimum votes
    bool allTied = true;
    int firstCount = voteCount.begin()->second;
    for (auto& [candidate, count] : voteCount)
    {
        if (count != firstCount)
        {
            allTied = false;
            break;
        }
    }

    if (allTied)
    {
        return { -1, true }; // All remaining candidates are tied
    }

    for (auto& [candidate, count] : voteCount)
    {
        if (count == minVotes)
        {
            eliminated.insert(candidate);
        }
    }

    // No winner yet, continue to next round
    return { -1, false };
}

将军 (Check the Check)

PC/UVa 题号: 110107/10196

你的任务是写一个程序，读入一个国际象棋棋局并且判断棋盘上的王是否被攻击(被将军)。如果王所处的位置能在一步之内被对手占领，我们称这个王被“将军”了。

大写字母代表白子，小写字母代表黑子。白方总是位于棋盘下部，而黑方总是位于棋盘上部。

如果你不熟悉国际象棋，下面是不同棋子的走子方式：

  • 卒 (p 或 P)：笔直向前移动，一次一格。但在吃子时，只能吃掉斜前方一格的棋子。在这个问题中你只用考虑吃子的情况。
  • 马 (n 或 N)：像中国象棋的马一样走“日”字形。只有马可以越过其他棋子 (没有中国象棋中蹩腿的规则)。
  • 象 (b 或 B)：可以沿对角线向前、向后走任意格。
  • 车 (r 或 R)：可以沿水平或竖直线向前、向后走任意格。
  • 后 (q 或 Q)：可以沿水平、竖直线或者对角线向任意方向走任意格。
  • 王 (k 或 K)：可以沿水平、竖直线或者对角线向任意方向走一格。

const int kingDirs[8][2] = { {-1, -1}, {-1, 0}, {-1, 1},

                              {1, -2},  {1, 2},  {2, -1},  {2, 1} };

// Directions for bishop/queen diagonal moves
const int diagDirs[4][2] = { {-1, -1}, {-1, 1}, {1, -1}, {1, 1} };


// Directions for rook/queen straight moves
const int straightDirs[4][2] = { {-1, 0}, {1, 0}, {0, -1}, {0, 1} };

/**
 * @brief Check if a position is within the 8x8 chessboard
 */
bool isValid(int x, int y) {
    return x >= 0 && x < 8 && y >= 0 && y < 8;
}


/**
 * @brief Check if the white king is in check by any black piece
 */

bool isWhiteKingInCheck(const vector& board, int wkX, int wkY) {
    // Check all 8 knight moves
    for (auto& dir : knightDirs) {
        int x = wkX + dir[0];
        int y = wkY + dir[1];
        if (isValid(x, y) && board[x][y] == 'n') {
            return true;
        }
    }

    // Check pawn attack (black pawns attack up-left/up-right)
    if (isValid(wkX - 1, wkY - 1) && board[wkX - 1][wkY - 1] == 'p') return true;
    if (isValid(wkX - 1, wkY + 1) && board[wkX - 1][wkY + 1] == 'p') return true;

    //Check straight/rook/queen attacks
    for (auto& dir : straightDirs) {
        int x = wkX + dir[0];
        int y = wkY + dir[1];
        while (isValid(x, y)) {
            char c = board[x][y];
            if (c != '.') {
                if (c == 'r' || c == 'q') return true;
                break;
            }
            x += dir[0];
            y += dir[1];
        }
    }

    // Check diagonal/bishop/queen attacks
    for (auto& dir : diagDirs) {
        int x = wkX + dir[0];
        int y = wkY + dir[1];
        while (isValid(x, y)) {
            char c = board[x][y];
            if (c != '.') {
                if (c == 'b' || c == 'q') return true;
                break;
            }
            x += dir[0];
            y += dir[1];
        }
    }

    // Check king attack (black king adjacent)
    for (auto& dir : kingDirs) {
        int x = wkX + dir[0];
        int y = wkY + dir[1];
        if (isValid(x, y) && board[x][y] == 'k') {
            return true;
        }
    }

    return false;
}

解释器 (Interpreter)

PC/UVa 题号: 110106/10033

某台电脑有 10 个寄存器和 1000 个内存单元的存储器。每一个寄存器或者内存单元可以保存一个 0 到 999 的三位十进制整数。指令被编码为三位十进制数字并存储在内存单元中。编码规则如下：

指令格式 功能说明
100 表示停机
2dn 表示将寄存器 d 的值设为 n(0 到 9)
3dn 表示给寄存器 d 的值增加 n
4dn 表示给寄存器 d 的值乘以 n
5ds 表示将寄存器 s 的值拷贝到寄存器 d
6ds 表示将寄存器 s 的值加到寄存器 d
7ds 表示将寄存器 s 的值乘到寄存器 d
8da 表示将地址为“寄存器 a 的值”的内存单元的值拷贝到寄存器 d
9sa 表示将寄存器 s 的值拷贝到地址为“寄存器 a 的值”的内存单元
0ds 表示若寄存器 s 的值不为 0，则跳转到地址为“寄存器 d 的值”的内存单元处继续执行

所有寄存器初始值都为 000。存储器的初始值将从输入数据读入。第一条指令总是在存储器地址 0 处执行。所有运算结果都为除以 1000 后的余数。

输入

输入第一行仅为一个正整数 n，表示测试数据组数。然后是一个空行，接下来是数据。每两组测试数据间均有一个空行。

每组测试数据包含最多 1000 个三位非负整数，代表从地址 0 开始的存储器的值。未指定的其他内存单元初始化为 000。

输出

对于每行测试数据输出一个整数：停机之前执行的总指令数 (包括停机指令本身)。假设程序总是会终止。相邻两组数据间用单个空行隔开。

样例输入

1 299 492 495 399 492 495 399 283 279 689 078 100 000 000 000

样例输出

16

#include <algorithm>
#include <iomanip>  //for setw(), setfill()(formatting debug output)
#include <iostream>
#include <vector>

using namespace  std;

/*
 *virtual machine interpreter for the problem simulates a computer
 *with 10 registers and 1000 memory celles
*/

class VMInterpreter
{
private:
    vector<int> registers;  //10 registers(0-9), initialized to 0
    vector<int> memory;   //1000 memory cells(0-999), initialized to 0
    int pc; //program counter(current instruction address)
    int instructionCount;   //number of instructions executed

    bool debugMode;

    //print current VM state(registers, PC, current instruction) for debug
    void printDebugState(int currentInstr)
    {
        if (!debugMode) return; //skip if debug mode is disabled

        cout << "=============== DEBUG STEP ===============" << endl;
        cout << "Program Counter(PC): " << setw(3) << setfill('0') << pc << endl;
        cout << "Current Instruction: " << setw(3) << setfill('0') << currentInstr << endl;
        cout << "Registers (0-9): ";
        for (int i = 0; i < 10; i++)
        {
            cout << "R" << i << "=" << setw(3) << setfill('0') << registers[i] << " ";
        }
        cout << endl;
        cout << "Memory (0-19): ";
        for (int i = 0; i < 19; ++i) {
            cout << "M" << setw(3) << setfill('0') << i << "=" << setw(3) << setfill('0') << memory[i] << " ";

            if ((i + 1) % 5 == 0) cout << endl;  //line break every 5 cells for readability
        }

        cout << endl;
        cout << "-----------------------------------------" << endl;
    }

    /*
     *decode and execute a single 3-digit instruction
    */
    bool executeInstruction(int instr)
    {
        printDebugState(instr);

        instructionCount++;
        int op = instr / 100;   //extract opcode(first digit)
        int d = (instr / 10) % 10;  //extract d parameter(second digit)
        int n = instr % 10;      //extract n parameter(third digit)
        int s = n;  //for instruction using s instead of n

        if (debugMode) // Debug: Print instruction decode result
        {
            cout << "Decode: opcode= " << op << ", d= " << d << ", n/s: " << n << endl;
        }

        bool halt = false;

        switch (op)
        {
        case 1:   //100: halt
            if (debugMode) cout << "Executing: HALT (program terminates)." << endl;
            halt = true;
            return true;

        case 2:   //2dn: set register d to n
            if (debugMode) cout << "Executing: SET R " << d << " = " << n << endl;
            registers[d] = n;
            break;

        case 3:   //3dn: add n to register d(mod 1000)
            if (debugMode) cout << "Executing: ADD " << n << " to R " << d << " (mod 1000)" << endl;
            registers[d] = (registers[d] + n) % 1000;
            break;

        case 4:   //4dn: multiply register d by n(mod 1000)
            if (debugMode) cout << "Executing: MUL " << n << " to R " << d << " (mod 1000)" << endl;
            registers[d] = (registers[d] * n) % 1000;
            break;

        case 5:   //5ds: copy register s to d
            if (debugMode) cout << "Executing: COPY R " << s << " to R " << d << endl;
            registers[d] = registers[s];
            break;

        case 6:   //6ds: add register s to d(mod 1000)
            if (debugMode) cout << "Executing: ADD R " << s << " to R " << d << " (mod 1000)" << endl;
            registers[d] = (registers[d] + registers[s]) % 1000;
            break;

        case 7:  //7ds: multiply register d by s(mod 1000)
            if (debugMode) cout << "Executing: MUL R " << s << " to R " << d << " (mod 1000)" << endl;
            registers[d] = (registers[d] * registers[s]) % 1000;
            break;

        case 8:  //8da: load from memory[reg[a]] to d
            if (debugMode) cout << "Executing: LOAD M[R " << n << " ] (" << memory[registers[n]] << ") to R " << d << endl;
            registers[d] = memory[registers[n]];
            break;

        case 9: //9sa: store register s to memory[reg[a]]
            if (debugMode) cout << "Executing: STORE R " << s << " (" << registers[s] << ") to M[R " << n << " ]" << endl;
            memory[registers[n]] = registers[s];
            break;

        case 0: //0ds: jump to memory[reg[d]] if reg[s] != 0
            if (registers[s] != 0)
            {
                if (debugMode) cout << "Executing: JUMP to M[R " << d << " ] (" << registers[d] << ") (R " << s << " != 0)" << endl;
                pc = registers[d] - 1; //adjust for pc increment after execute
            }
            else
            {
                if (debugMode) cout << "Executing: JUMP skipped (R " << s << " == 0)" << endl;
            }
            break;

        default:
            //invalid opcode, do nothing
            if (debugMode) cout << "Executing: INVALID OPCODE (" << op << ") - do nothing" << endl;
            break;
        }

        // Print instruction count after execution
        if (debugMode) {
            cout << "Instruction Count so far: " << instructionCount << endl;
            cout << "============================================" << endl << endl;
        }
        return halt;
    }

public:
    VMInterpreter(bool debug = true) : registers(10, 0),
        memory(1000, 0), pc(0), instructionCount(0), debugMode(debug) {
    }

    bool isDebugModeEnabled() const {
        return debugMode;
    }

    //load initial memory data into the VM
    void loadMemory(const vector<int>& data)
    {
        fill(registers.begin(), registers.end(), 0);
        fill(memory.begin(), memory.end(), 0);
        pc = 0;
        instructionCount = 0;

        //load input data into memory
        for (int i = 0; i < data.size(); i++)
        {
            memory[i] = data[i];
        }

        if (debugMode) {
            cout << "================= INIT LOAD ================" << endl;
            cout << "Loaded " << data.size() << " instructions into memory (address 0 to " << data.size() - 1 << ")" << endl;
            cout << "Initial Memory (0 to " << min(19, (int)data.size() - 1) << "): ";
            for (int i = 0; i < min(20, (int)data.size()); ++i) {
                cout << "M" << setw(3) << setfill('0') << i << "=" << setw(3) << setfill('0') << memory[i] << " ";
                if ((i + 1) % 5 == 0) cout << endl;
            }
            cout << endl;
            cout << "============================================" << endl << endl;
        }
    }

    //run the VM until halt instruction is executed
    //return number of instructions executed(including halt)
    int run()
    {
        if (debugMode) {
            cout << "================ START RUN ================" << endl;
            cout << "VM starts executing from address 0" << endl;
            cout << "===========================================" << endl << endl;
        }

        while (true)
        {
            int instr = memory[pc];;
            pc++;
            if (executeInstruction(instr))
            {
                break;
            }
        }

        // Debug: Print final result
        if (debugMode) {
            cout << "================  FINISH RUN ================ " << endl;
            cout << "Program halted (opcode 100 detected)" << endl;
            cout << "Total Instructions Executed: " << instructionCount << endl;
            cout << "Final Registers (0-9): ";
            for (int i = 0; i < 10; ++i) {
                cout << "R" << i << "=" << setw(3) << setfill('0') << registers[i] << " ";
            }
            cout << endl;
            cout << "===============================================" << endl << endl;
        }
        return instructionCount;
    }

    /**
       * @brief Enable or disable debug mode dynamically
       * @param debug true = enable, false = disable
       */
    void setDebugMode(bool debug) {
        debugMode = debug;
    }
};

图形化编辑器 (Graphical Editor)

PC/UVa 题号: 110105/10267

正如文本编辑器允许我们修改文本一样，Photoshop 这样的图形化编辑器允许我们修改位图图像。在本题中，图像被表示成一个 M × N 的像素数组，每个像素有一种颜色。你的任务是编写一个程序，模拟一个简单的交互式图形化编辑器。

输入

输入包括一个指令序列，每条指令占单独的一行。每行的第一个字符总是大写字母，代表这条指令的类型。如果该指令需要参数，将会在同一行中陆续给出，用空格隔开。

像素坐标用列号和行号这两个整数表示，列号在 1 到 M 范围内，行号在 1 到 N 范围内 (1 ≤ M,N ≤ 250)。坐标原点在左上角。颜色用大写字母表示。

编辑器接受以下命令：

命令格式 功能说明
I M N 创建一个新的 M × N 图像，所有像素为白色 (0)。
C 清除图像，把所有像素涂成白色 (0)，图像大小不变。
L X Y C 把像素 (X,Y) 涂成颜色 (C)。
V X Y1 Y2 C 画一条竖直线段，颜色为 (C)，列号为 X，从第 Y1 行到第 Y2 行 (包括 Y1 和 Y2 行)。
H X1 X2 Y C 画一条水平线段，颜色为 (C)，行号为 Y，从第 X1 列到第 X2 列 (包括 X1 和 X2 列)。
K X1 Y1 X2 Y2 C 画一个实心矩形，颜色为 (C)，左上角为 (X1,Y1)，右下角为 (X2,Y2)。
F X Y C 把区域 R 填充为颜色 (C)。R 定义如下：若像素 (X,Y) 属于 R，则所有与 (X,Y) 有公共边且颜色相同的像素也属于 R。
S NAME 原样输出文件名以及整幅图像的像素矩阵。
X 退出程序。

输出

对于每条S NAME指令，原样输出文件名NAME和当前图像的内容。输出的每一行表示图像每一行中各个像素的颜色，参见样例输出。

如果指令的第一个字符不是I、C、L、V、H、K、F、S、X之一，忽略此命令，直接跳到下一行输入进行处理。

I 5 6 L 2 3 A S one.bmp G 2 3 J F 3 3 J V 2 3 4 W H 3 4 2 Z S two.bmp X

one.bmp 00000 00000 0A000 00000 00000 00000 two.bmp JJJJJ JJZZJ JWJJJ JWJJJ JJJJJ JJJJJ

#include <iostream>
#include <vector>
#include <algorithm>
#include <string>

using namespace  std;

/*
 * CanvasEditor class encapsulates all functionalities of a simple graphic editor
 * Manages canvas state and provides methods for  drawing , filling and output
 *
 */

class CanvasEditor
{
private:
    vector<vector<char>> canvas;  //2d vector to store pixel data of the canvas
    int cols;
    int rows;;

    void floodFillHelper(int x, int y, char target, char color)
    {
        //convert 1-based coordinates to 0-based for vector access
        int cx = x - 1;
        int cy = y - 1;;

        //boundary check: if coordinates are out of canvas range, return
        if (cx < 0 || cx >= cols || cy < 0 || cy >= rows)
        {
            return;
        }

        if (canvas[cy][cx] != target)
        {
            return;
        }

        //fill current pixel with new color
        canvas[cy][cx] = color;

        //recursively fill 4 adjacent directions(right, left, down, up)
        floodFillHelper(x + 1, y, target, color); //right
        floodFillHelper(x - 1, y, target, color); //left
        floodFillHelper(x, y + 1, target, color); //down
        floodFillHelper(x, y - 1, target, color); //up
    }

public:
    CanvasEditor() : cols(0), rows(0) {};

    void createCanvas(int M, int N)
    {
        cols = M;
        rows = N;
        canvas.assign(rows, vector<char>(cols, '0'));
    }

    void clearCanvas()
    {
        if (canvas.empty())
        {
            return;
        }

        for (int i = 0; i < rows; ++i)
        {
            fill(canvas[i].begin(), canvas[i].end(), '0');
        }
    }

    void drawPixel(int X, int Y, char C)
    {
        if (X < 1 || X > cols || Y < 1 || Y > rows)
        {
            return;
        }

        canvas[Y - 1][X - 1] = C;
    }

    void drawVerticalLine(int X, int Y1, int Y2, char C)
    {
        if (X < 1 || X > cols)
            return;

        if (Y1 > Y2)
            swap(Y1, Y2);

        for (int y = Y1; y <= Y2; ++y)
        {
            canvas[y - 1][X - 1] = C;
        }
    }

    void drawHorizontalLine(int X1, int X2, int Y, char C)
    {
        if (Y < 1 || Y > rows)
            return;

        if (X1 > X2)
            swap(X1, X2);

        for (int x = X1; x <= X2; ++x)
        {
            canvas[Y - 1][x - 1] = C;
        }
    }

    void drawRectangle(int X1, int Y1, int X2, int Y2, char C)
    {
        if (X1 > X2)
            swap(X1, X2);

        if (Y1 > Y2)
            swap(Y1, Y2);

        if (X1 < 1 || X2 > cols || Y1 < 1 || Y2 > rows)
        {
            return;
        }

        //fill all pixels in the rectangle area
        for (int y = Y1; y <= Y2; ++y)
        {
            for (int x = X1; x <= X2; ++x)
            {
                canvas[y - 1][x - 1] = C;
            }
        }
    }

    void fillRegion(int X, int Y, char C)
    {
        if (X < 1 || X > cols || Y < 1 || Y > rows)
            return;

        char target = canvas[Y - 1][X - 1];

        //avoid infinite recursion if target color == fill color
        if (target != C)
        {
            floodFillHelper(X, Y, target, C);
        }
    }

    void printCanvas(const string& name)
    {
        cout << name << endl;

        for (int i = 0; i < rows; ++i)
        {
            for (int j = 0; j < cols; ++j)
            {
                cout << canvas[i][j];
            }
            cout << endl;
        }
    }
};

int main()
{
    CanvasEditor editor;
    char cmd;

    while (cin >> cmd)
    {
        switch (cmd)
        {
        case 'I':
        {
            int M, N;
            cin >> M >> N;
            editor.createCanvas(M, N);
            break;
        }

        case 'C':
            editor.clearCanvas();
            break;

        case 'L':
        {
            int X, Y;
            char C;
            cin >> X >> Y >> C;
            editor.drawPixel(X, Y, C);
            break;
        }

        case 'V':
        { // Command: Draw vertical line
            int X, Y1, Y2;
            char C;
            cin >> X >> Y1 >> Y2 >> C;
            editor.drawVerticalLine(X, Y1, Y2, C);
            break;
        }

        case 'H':
        { // Command: Draw horizontal line
            int X1, X2, Y;
            char C;
            cin >> X1 >> X2 >> Y >> C;
            editor.drawHorizontalLine(X1, X2, Y, C);
            break;
        }

        case 'K':
        { // Command: Draw solid rectangle
            int X1, Y1, X2, Y2;
            char C;
            cin >> X1 >> Y1 >> X2 >> Y2 >> C;
            editor.drawRectangle(X1, Y1, X2, Y2, C);
            break;
        }

        case 'F':
        { // Command: Flood fill region
            int X, Y;
            char C;
            cin >> X >> Y >> C;
            editor.fillRegion(X, Y, C);
            break;
        }

        case 'S':
        { // Command: Print canvas
            string name;
            cin >> name;
            editor.printCanvas(name);
            break;
        }

        case 'X':
            return 0;

        default:
            cin.ignore(1024, '\n');
            break;
        }
    }

    return 0;
}

// PC/UVa 110104/706 - 液晶显示屏 (LC-Display)
#include <iostream>
#include <string>
#include <vector>
#include <span>

/*
 *brief define the segment structure of each digit(0 - 9)
 *each digit is represented by 7 segments(a-g), where:
 *a: top horizontal, b : top-right vertical, c: bottom-right vertical,
 *d: bottom horizontal, e : bottom-left vertical, f : top-left vertical,
 *g: middle horizontal
 */
const bool DIGIT_SEGMENTS[10][7] = {
    {1,1,1,1,1,1,0},  //0: a, b, c, d, e, f
    {0,1,1,0,0,0,0},  //1: b, c
    {1,1,0,1,1,0,1},  //2: a, b, g, d, e
    {1,1,1,1,0,0,1},  //3: a, b, g, c, d
    {0,1,1,0,0,1,1},  //4: f, b, g, c
    {1,0,1,1,0,1,1},  //5: a, f, g, c, d
    {1,0,1,1,1,1,1},  //6: a, f, g, c, d, e
    {1,1,1,0,0,0,0},  //7: a, b, c
    {1,1,1,1,1,1,1},  //8: all segments
    {1,1,1,1,0,1,1}   //9: a, b, g, c, d, f
};

/* generates the LCD-style representation of a single digit */
std::vector<std::string> generateDigit(int s, int digit)
{
    const bool *seg = DIGIT_SEGMENTS[digit];
    std::vector<std::string> lines(2 * s + 3);
    int cols = s + 2;

    //line 0 : top horizontal (segment a)
    lines[0] = " " + (seg[0] ? std::string(s, '-') : std::string(s, ' ')) + " ";
    //lines 1~s: top verticals (segments f, b)
    for (int i = 1; i <= s; ++i)
    {
        lines[i] = (seg[5] ? "|" : " ") + std::string(s, ' ') + (seg[1] ? "|" : " ");
    }
    //line s+1: middle horizontal (segment g)
    lines[s+1] = " " +  (seg[6] ? std::string(s, '-') : std::string(s, ' ')) + " ";
    //lines s+2 ~ 2s+1: bottom verticals (segments e, c)
    for (int i = s+2; i <= 2*s +1; ++i)
    {
        lines[i] = (seg[4] ? "|" : " ") + std::string(s, ' ') + (seg[2] ? "|" : " ");
    }
    //line 2s+2: bottom horizontal (segment d)
    lines[2*s+2] = " " + (seg[3] ? std::string(s, '-') : std::string(s, ' ')) + " ";

    return lines;
}

//combines multiple digit representation into a single LCD line
std::vector<std::string> combineDigits(int s, std::span<const std::vector<std::string>> digits)
{
    std::vector<std::string> result(2*s + 3);
    for (size_t line = 0; line < result.size(); ++line)
    {
        for (size_t i = 0; i < digits.size(); ++i)
        {
            result[line] += digits[i][line];
            if (i != digits.size()-1)
            {
                result[line] += " ";  //add one space between digits
            }
        }
    }
    return result;
}

int main()
{
    std::ios::sync_with_stdio(false);  //optimize IO speed
    std::cin.tie(nullptr);

    int s;
    std::string n_str;
    while (std::cin >> s >> n_str && !(s == 0 && n_str == "0"))
    {
        //generate each digit's LCD representation
        std::vector<std::vector<std::string>> digit_lines;
        for (char c : n_str)
        {
            int digit = c - '0';
            digit_lines.push_back(generateDigit(s, digit));
        }

        //combine digits and output the LCD result
        std::vector<std::string> display = combineDigits(s, digit_lines);
        for (const auto& line : display)
        {
            std::cout << line << std::endl;
        }
        std::cout << std::endl;  //empty line after each number
    }
    return 0;

/**
 * @brief 计算平摊费用所需的最小总交易金额（以分为单位）
 * 
 * 核心逻辑：将所有支出转换为分避免浮点精度问题，计算平均支出后，
 * 统计所有支出低于平均值的学生需要补的总金额（即为最小交易总额）
 * 
 * @param n 学生人数
 * @param cents 每个学生的支出（以分为单位的数组）
 * @return long long 最小总交易金额（分）
 */
long long calculateMinTranscaction(int n, std::span<const int> cents)
{
    long long total = 0;
    for (int c : cents)
    {
        total += c;
    }

    const int avg = total / n;
    long long transfer = 0;

    for (int c : cents)
    {
        if (c < avg)
        {
            transfer += avg - c;
        }
    }

    return transfer;
}

static std::mt19937 rng(std::random_device{}());

/**
 * @brief Counts the number of mines in the 8 adjacent cells of the target cell (i, j).
 *
 * This function checks all 8 surrounding cells (up, down, left, right, and four diagonals)
 * of the specified cell (i, j). It skips the target cell itself and only counts valid cells
 * within the grid bounds to avoid out-of-bounds access.
 *
 * @param grid Constant reference to the input mine grid, stored as a vector of strings.
 * @param n Total number of rows in the mine grid.
 * @param m Total number of columns in the mine grid.
 * @param i Row index of the target cell to check.
 * @param j Column index of the target cell to check.
 * @return int The total number of adjacent mines (range from 0 to 8).
 */
int countAdjacentMines(const std::vector<std::string>& grid, int n, int m, int i, int j) {
    int count = 0;
    // Iterate through all 8 adjacent directions (di = -1, 0, 1; dj = -1, 0, 1)
    for (int di = -1; di <= 1; ++di) {
        for (int dj = -1; dj <= 1; ++dj) {
            // Skip the current target cell (no need to count itself as a mine)
            if (di == 0 && dj == 0) continue;
            // Calculate the coordinates of the adjacent cell
            int ni = i + di;
            int nj = j + dj;
            // Check if the adjacent cell is within the valid grid boundaries
            if (ni >= 0 && ni < n && nj >= 0 && nj < m) {
                // Increment count if the adjacent cell contains a mine ('*')
                if (grid[ni][nj] == '*') {
                    ++count;
                }
            }
        }
    }
    return count;
}

/**
 * @brief Generates a single random mine grid with specified dimensions and mine density.
 *
 * This function initializes a grid filled with safe cells ('.') first, then randomly
 * sets cells to mines ('*') based on the given mine density probability. It uses the
 * global unified random number generator to ensure diverse and non-repetitive results.
 *
 * @param n Number of rows in the generated mine grid.
 * @param m Number of columns in the generated mine grid.
 * @param mineDensity Probability of a cell being a mine (range from 0.0 to 1.0, e.g., 0.2 = 20% mines).
 * @return std::vector The generated mine grid, each string represents a single row of the grid.
 */
std::vector<std::string> generateMineGrid(int n, int m, double mineDensity) {
    // Bernoulli distribution to generate boolean values (true for mine, false for safe cell)
    std::bernoulli_distribution dist(mineDensity);

    // Initialize grid with all safe cells ('.')
    std::vector<std::string> grid(n, std::string(m, '.'));
    for (int i = 0; i < n; ++i) {
        for (int j = 0; j < m; ++j) {
            // Randomly set the current cell to a mine based on the density distribution
            if (dist(rng)) {
                grid[i][j] = '*';
            }
        }
    }
    return grid;
}

uint64_t collatzCycleLength(uint64_t n) {
    uint64_t count = 1;
    while (n != 1) {
        if (n % 2 == 0) {
            n /= 2;
        }
        else
        {
            n = 3 * n + 1;
        }
        count++;
    }
    return count;
}
