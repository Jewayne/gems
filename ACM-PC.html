PC/UVa 110102/10189 - 扫雷 (Minesweeper)

题目描述

你玩过扫雷游戏吗？这个有趣的小游戏来自于某个快被忘却的操作系统。游戏的目标是找出一个 n×m 矩阵内的所有地雷。在本题中，你需要为每个单元格统计出它周围的地雷个数。每个单元格最多有 8 个相邻的单元格。

输入

输入将包含若干个矩阵。对于每一个矩阵，第一行包含两个数字 n 和 m (0 < n,m ≤ 100)，分别代表这个矩阵的行数和列数。接下来的 n 行每行包含 m 个字符，即该矩阵。安全格子用 "." 表示，有地雷的格子用 "*" 表示。当 n = m = 0 时，表示输入结束。你的程序不应处理这一行。

输出

对于每一个矩阵，首先在单独的一行里打印序号：Field #x：其中 x 是数据编号，从 1 开始。接下来的 n 行中，读入的 "." 应该被该位置周围的地雷数所代替。输出的每两个相邻矩阵必须用一个空行隔开。

样例输入

4 4
*...
....
.*..
....
3 5
**...
.....
.*...
0 0

样例输出

Field #1:
*100
2210
1*10
1110

Field #2:
**100
33200
1*100

解题代码

// PC/UVa 110102/10189 Minesweeper
#include <iostream>
#include <vector>
#include <string>
#include <fstream>
#include <stdexcept>
#include <random>
#include <algorithm>
#include <span>  // C++20: Reduces copy operations and improves access efficiency

// Global unified random number generator (avoids duplicate definition and ensures random consistency)
static std::mt19937 rng(std::random_device{}());

/**
 * @brief Counts the number of mines in the 8 adjacent cells of the target cell (i, j).
 *
 * This function checks all 8 surrounding cells (up, down, left, right, and four diagonals)
 * of the specified cell (i, j). It skips the target cell itself and only counts valid cells
 * within the grid bounds to avoid out-of-bounds access.
 *
 * @param grid Constant reference to the input mine grid, stored as a vector of strings.
 * @param n Total number of rows in the mine grid.
 * @param m Total number of columns in the mine grid.
 * @param i Row index of the target cell to check.
 * @param j Column index of the target cell to check.
 * @return int The total number of adjacent mines (range from 0 to 8).
 */
int countAdjacentMines(const std::vector<std::string>& grid, int n, int m, int i, int j) {
    int count = 0;
    // Iterate through all 8 adjacent directions (di = -1, 0, 1; dj = -1, 0, 1)
    for (int di = -1; di <= 1; ++di) {
        for (int dj = -1; dj <= 1; ++dj) {
            // Skip the current target cell (no need to count itself as a mine)
            if (di == 0 && dj == 0) continue;
            // Calculate the coordinates of the adjacent cell
            int ni = i + di;
            int nj = j + dj;
            // Check if the adjacent cell is within the valid grid boundaries
            if (ni >= 0 && ni < n && nj >= 0 && nj < m) {
                // Increment count if the adjacent cell contains a mine ('*')
                if (grid[ni][nj] == '*') {
                    ++count;
                }
            }
        }
    }
    return count;
}

/**
 * @brief Generates a single random mine grid with specified dimensions and mine density.
 *
 * This function initializes a grid filled with safe cells ('.') first, then randomly
 * sets cells to mines ('*') based on the given mine density probability. It uses the
 * global unified random number generator to ensure diverse and non-repetitive results.
 *
 * @param n Number of rows in the generated mine grid.
 * @param m Number of columns in the generated mine grid.
 * @param mineDensity Probability of a cell being a mine (range from 0.0 to 1.0, e.g., 0.2 = 20% mines).
 * @return std::vector The generated mine grid, each string represents a single row of the grid.
 */
std::vector<std::string> generateMineGrid(int n, int m, double mineDensity) {
    // Bernoulli distribution to generate boolean values (true for mine, false for safe cell)
    std::bernoulli_distribution dist(mineDensity);

    // Initialize grid with all safe cells ('.')
    std::vector<std::string> grid(n, std::string(m, '.'));
    for (int i = 0; i < n; ++i) {
        for (int j = 0; j < m; ++j) {
            // Randomly set the current cell to a mine based on the density distribution
            if (dist(rng)) {
                grid[i][j] = '*';
            }
        }
    }
    return grid;
}

/**
 * @brief Automatically generates multiple mine grid samples and saves them to testin.txt.
 *
 * This function generates a specified number of mine grids with random dimensions (within the set range),
 * writes them to the testin.txt file in the required format, and appends the termination line "0 0" at the end.
 * It also provides a user prompt before overwriting the existing file (if any).
 */
void outputGeneratedSampleTestin() {
    const int numFields = 5;          // Total number of mine grids to generate
    const int minSize = 3;            // Minimum number of rows/columns for each grid
    const int maxSize = 10;           // Maximum number of rows/columns for each grid
    const double mineDensity = 0.2;   // Probability of each cell being a mine (20%)
    const std::string outputFileName = "testin.txt"; // Output file name for generated samples

    // Inform user that the existing file will be overwritten if it exists
    std::cout << "Info: Preparing to generate mine samples, will overwrite \"" << outputFileName << "\" if it exists." << std::endl;
    // Open the output file in write mode (creates the file if it does not exist)
    std::ofstream outputFile(outputFileName);
    if (!outputFile.is_open()) {
        std::cerr << "Error: Can't create output file \"" << outputFileName << "\"." << std::endl;
        return;
    }

    // Uniform distribution to generate random grid dimensions (n and m)
    std::uniform_int_distribution<int> sizeDist(minSize, maxSize);

    // Generate each mine grid and write to the output file
    for (int i = 0; i < numFields; ++i) {
        // Generate random row and column counts for the current grid
        int n = sizeDist(rng);
        int m = sizeDist(rng);

        // Write the grid dimensions to the file (first line of each field)
        outputFile << n << " " << m << std::endl;

        // Generate the mine grid and write each row to the file
        std::vector<std::string> grid = generateMineGrid(n, m, mineDensity);
        for (const auto& row : grid) {
            outputFile << row << std::endl;
        }
    }

    // Write the termination line to signal the end of input (required by the mine sweeper logic)
    outputFile << "0 0" << std::endl;

    // Close the output file to release system resources
    outputFile.close();
    std::cout << "Successfully generated " << numFields << " mine grids." << std::endl;
    std::cout << "Result saved to \"" << outputFileName << "\"." << std::endl;
}

/**
 * @brief Outputs the processed mine grid result to both console and testout.txt with unified format.
 *
 * This function ensures the output format is consistent between the console and the output file,
 * including the "Field #x:" header and the processed grid content. It reuses the same result data
 * to avoid redundant calculations and ensure data consistency.
 *
 * @param outputFile Reference to the ofstream object for writing to testout.txt.
 * @param result Constant reference to the processed result grid (mine counts and preserved mines).
 * @param fieldNum The sequential number of the current field (for formatting the header).
 */
void outputResult(std::ofstream& outputFile, const std::vector<std::string>& result, int fieldNum) {
    // Output the result to the console (user-friendly real-time feedback)
    std::cout << "Field #" << fieldNum << ":" << std::endl;
    for (const auto& row : result) {
        std::cout << row << std::endl;
    }

    // Output the same result to testout.txt (for permanent storage and later verification)
    outputFile << "Field #" << fieldNum << ":" << std::endl;
    for (const auto& row : result) {
        outputFile << row << std::endl;
    }
}

/**
 * @brief Main function: Orchestrates the entire process (generate input → process → double output).
 *
 * This function coordinates the full workflow: first generates the input sample (testin.txt),
 * then reads the input file, processes each mine grid to count adjacent mines, and finally
 * outputs the result to both the console and testout.txt. It also includes error handling
 * for file operations to ensure robust execution.
 *
 * @return int Program exit code (0 for successful execution, 1 for critical errors).
 */
int main() {
    // Define fixed input and output file paths (required by the task requirements)
    const std::string inputFileName = "testin.txt";
    const std::string outputFileName = "testout.txt";

    // Step 1: Automatically generate the input mine samples (save to testin.txt)
    outputGeneratedSampleTestin();

    // Step 2: Open the input file (read-only) and output file (write-only, overwrite existing)
    std::ifstream inputFile(inputFileName);
    std::ofstream outputFile(outputFileName);

    // Step 3: Validate if both files are opened successfully (error handling for file operations)
    if (!inputFile.is_open()) {
        std::cerr << "Error: Cannot open input file \"" << inputFileName << "\"." << std::endl;
        std::cerr << "Please make sure the file exists and is in the correct directory." << std::endl;
        return 1;
    }
    if (!outputFile.is_open()) {
        std::cerr << "Error: Cannot create or open output file \"" << outputFileName << "\"." << std::endl;
        inputFile.close(); // Close the input file before exiting to avoid resource leaks
        return 1;
    }

    int n, m;                // Dimensions of the current mine grid (rows, columns)
    int fieldNum = 0;        // Sequential number of the current field (for output formatting)

    // Step 4: Read and process each mine grid from testin.txt until the termination line (0 0)
    while (inputFile >> n >> m && (n != 0 || m != 0)) {
        // Print an empty line between fields (except for the first field) to maintain format consistency
        if (fieldNum > 0) {
            std::cout << std::endl;
            outputFile << std::endl; // Sync the empty line to the output file
        }
        fieldNum++;

        // Ignore the remaining newline character after reading n and m to avoid empty line issues with getline
        inputFile.ignore();

        // Step 5: Read the entire mine grid from the input file
        std::vector<std::string> grid(n);
        for (int i = 0; i < n; ++i) {
            std::getline(inputFile, grid[i]);
            // Validate the length of each row to ensure it matches the column count m (avoid format errors)
            if (grid[i].length() != static_cast<size_t>(m)) {
                std::cerr << "Error: Line " << i + 1 << " of field " << fieldNum << " has invalid length." << std::endl;
                // Close both files before exiting to release system resources and avoid leaks
                inputFile.close();
                outputFile.close();
                return 1;
            }
        }

        // Step 6: Process the mine grid to generate the result (count adjacent mines for safe cells)
        std::vector<std::string> result(n, std::string(m, ' '));
        for (int i = 0; i < n; ++i) {
            // C++20: Use std::span to create a non-copy view of the grid row for improved access efficiency
            std::span<const char> grid_row_span(grid[i].c_str(), m);
            for (int j = 0; j < m; ++j) {
                if (grid_row_span[j] == '*') {
                    // Preserve the mine cell ('*') in the result grid without modification
                    result[i][j] = '*';
                }
                else {
                    // Count adjacent mines and convert the count to a character (0-8) for the result grid
                    int mineCount = countAdjacentMines(grid, n, m, i, j);
                    result[i][j] = '0' + mineCount;
                }
            }
        }

        // Step 7: Output the processed result to both console and testout.txt
        outputResult(outputFile, result, fieldNum);
    }

    // Step 8: Close both files to release system resources and avoid resource leaks
    inputFile.close();
    outputFile.close();

    // Step 9: Notify the user that the entire process has completed successfully
    std::cout << "\n\nProcess completed successfully." << std::endl;
    std::cout << "Result has been saved to \"" << outputFileName << "\"." << std::endl;
    return 0;
}

代码说明

1. 核心函数：countAdjacentMines 遍历目标单元格8个相邻方向，统计地雷数量，严格校验边界避免越界；

2. 测试数据生成：generateMineGrid + outputGeneratedSampleTestin 自动生成随机扫雷矩阵并写入文件，方便测试；

3. 结果输出：outputResult 统一控制台和文件输出格式，保证样例要求的空行分隔、序号标注；

4. 鲁棒性：校验文件打开状态、矩阵行长度，避免格式错误导致程序崩溃；

5. 现代C++特性：使用 std::span 减少数据拷贝，std::random 生成随机测试用例，提升代码效率和可测试性。

PC/UVa 110101/100 - 3n+1 问题

题目描述

考虑如下的序列生成算法：从整数 n 开始，如果 n 是偶数，把它除以 2；如果 n 是奇数，把它乘 3 加 1。用新得到的值重复上述步骤，直到 n = 1 时停止。例如，n = 22 时该算法生成的序列是：

22, 11, 34, 17, 52, 26, 13, 40, 20, 10, 5, 16, 8, 4, 2, 1

人们猜想（没有得到证明）对于任意整数 n，该算法总能终止于 n = 1。这个猜想对于至少 1 000 000 内的整数都是正确的。

对于给定的 n，该序列的元素（包括 1）个数被称为 n 的循环节长度。在上述例子中，22 的循环节长度为 16。输入两个数 i 和 j，你的任务是计算 i 到 j（包含 i 和 j）之间的整数中，循环节长度的最大值。

输入

输入每行包含两个整数 i 和 j。所有整数大于 0，小于 1 000 000。

输出

对于每对整数 i 和 j，按原来的顺序输出 i 和 j，然后输出二者之间的整数中的最大循环节长度。这三个整数应该用单个空格隔开，且在同一行输出。对于读入的每一组数据，在输出中应位于单独的一行。

样例输入

1 10
100 200
201 210
900 1000

样例输出

1 10 20
100 200 125
201 210 89
900 1000 174

解题代码

// 3n+1 问题 - PC/UVa:11101/100
#include <iostream>
#include <algorithm> //for std::min , std::max
#include <cstdint>  //for fixed-size integer type to avoid overflow
#include <vector>
#include <chrono>
#include <iomanip>
#include <format>
#include <concepts>
#include <span>

//concept to constrain integer types(replaces std::is_integral + std::enable_if)
template<typename T>
concept IntegralType = std::is_integral_v<T> && std::is_unsigned_v<T>;

/*
performance test function: compare normal and memoized version
range: test interval(std::span for lightweight, no-copy range)
repeat: number for test repeats to get stable average time
*/
template<typename Func, IntegralType T>
double testCollatzPerfermance(Func func, std::span<const T>range, uint32_t repeat) noexcept {
    static_assert(std::is_invocable_r_v<uint64_t, Func, T>,
        "Function must return uint64_t and take T as parameter");

    //record start time(high_resolution_clock for maximum precision)
    auto start = std::chrono::high_resolution_clock::now();

    uint64_t final_dummy_max = 0;
    for (uint32_t r = 0; r < repeat; r++) {
        uint64_t dummy_max = 0;  //to avoid compiler optimization
        for (const auto& num : range) {
            uint64_t len = func(static_cast<uint64_t>(num));
            if (len > dummy_max) {
                dummy_max = len;
            }
        }
        final_dummy_max = dummy_max;
    }

    std::cout << std::format("Test dummy max: {}\n", final_dummy_max);

    auto end = std::chrono::high_resolution_clock::now();
    std::chrono::duration<double, std::milli> elapsed = end - start;

    return elapsed.count() / static_cast<double>(repeat);
}

uint64_t collatzCycleLength(uint64_t n) {
    uint64_t count = 1;
    while (n != 1) {
        if (n % 2 == 0) {
            n /= 2;
        }
        else
        {
            n = 3 * n + 1;
        }
        count++;
    }
    return count;
}

//optimized version with memoization to avoid redundant calculations
//cache stores computed cycle lengths to speed up repeated queries
constexpr uint64_t MAX_CACHE = 1000000;
uint64_t cache[MAX_CACHE + 1] = {0};

//memoized cycle length calculation
uint64_t collatzCycleLengthMemo(uint64_t n) {
    if (n <= MAX_CACHE && cache[n] != 0) {
        return cache[n];
    }

    uint64_t count = 1;
    uint64_t temp = n;
    while (temp != 1) {
        if (temp % 2 == 0) {
            temp /= 2;
        }
        else {
            temp = 3 * temp + 1;
        }
        count++;
    }
    if (n <= MAX_CACHE) {
        cache[n] = count;
    }
    return count;
}

int main()
{
    constexpr uint64_t TEST_LEFT = 1;
    constexpr uint64_t TEST_RIGHT = 1000000;
    constexpr uint64_t TEST_REPEAT = 5;

    std::vector<uint64_t> test_range;
    test_range.reserve(TEST_RIGHT - TEST_LEFT + 1);
    for (uint64_t num = TEST_LEFT; num <= TEST_RIGHT; num++) {
        test_range.push_back(num);
    }

    //wrap vector with std::span
    std::span<const uint64_t> test_span(test_range);

    double normal_avg_time = testCollatzPerfermance(collatzCycleLength, test_span, TEST_REPEAT);
    double memo_avg_time = testCollatzPerfermance(collatzCycleLengthMemo, test_span, TEST_REPEAT);

    std::cout << std::format("=== Collatz sequence Performance Test ===\n");
    std::cout << std::format("Test range: [{}, {}]\n", TEST_LEFT, TEST_RIGHT);
    std::cout << std::format("Test repeats: {}\n", TEST_REPEAT);
    std::cout << std::format("------------------------------------------\n");
    std::cout << std::format("Normal version average time: {:.3f} ms\n", normal_avg_time);
    std::cout << std::format("Memoized version average time: {:.3f} ms\n", memo_avg_time);
    std::cout << std::format("-------------------------------------------\n");
    std::cout << std::format("Performance speedup: {:.2f}x faster\n", normal_avg_time / memo_avg_time);

    std::cout << std::format("\n=== Traditional Collate Query ===\n");
    uint64_t i, j;
    while (std::cin >> i >> j) {
        const uint64_t original_i = i;
        const uint64_t original_j = j;
        const uint64_t left = std::min(i, j);
        const uint64_t right = std::max(i, j);

        uint64_t max_len = 0;
        for (uint64_t num = left; num <= right; num++)
        {
            const uint64_t len = collatzCycleLengthMemo(num);
            if (len > max_len) {
                max_len = len;
            }
        }
        std::cout << std::format("{} {} {}\n", original_i, original_j, max_len);
    }
    return 0;
}

代码说明

1. 核心函数：collatzCycleLength 计算单个数字的循环节长度，collatzCycleLengthMemo 增加缓存优化，避免重复计算；

2. 性能测试：testCollatzPerfermance 对比普通版和缓存版的执行效率，输出平均耗时和加速比；

3. 数据类型：使用 uint64_t 避免数值溢出，适配 100万 以内的计算；

4. 鲁棒性：兼容输入 i > j 的情况，通过 std::min/max 确定遍历范围；

5. 现代C++特性：使用 concept 约束类型、std::span 减少容器拷贝，提升性能。
